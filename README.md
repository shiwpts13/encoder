# encoder
## Installation in linux(Debian, ubuntu, kali):
    
* apt update
* sudo apt install python3
* sudo apt install python3-pip
* sudo apt install git
* git clone https://github.com/shiwpts13/encoder
* cd encoder
* python3 -m pip install colorama
* python3 menu.py
